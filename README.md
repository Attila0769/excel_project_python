Intallation : 

Make sure you have python installed with pip.
Download the project with one of the different options github offers. 
Create the venv environment : 
- Open a powershell terminal in the project folder :
- Run the command :
-   python -m venv .venv
- Then
-   pip install -r requirements.txt

Now you can run the python files without any problem with jupyter notebook, vs code with jupyter extension or any software to run .ipynb files. 

The plots created can all be found in the folder plots
carte_ventes.html shows a map of the US with the cities where the company sold the most in terms of revenue.
